# Agent Rules

## Core

- 항상 한국어로 답변한다.
- 사용하지 않는 코드는 삭제한다.
- 작업 시작 전에 이 파일과 작업 범위에 해당하는 `.agents/rules` 문서를 읽는다.

## Rule Routing

- 모든 작업: [project.md](.agents/rules/project.md), [workflow.md](.agents/rules/workflow.md)
- 플러그인 또는 Rails Engine 작업: [plugins.md](.agents/rules/plugins.md)
- UI, ERB, Tailwind CSS 작업: [frontend.md](.agents/rules/frontend.md)

여러 범위에 걸친 작업은 해당 규칙 문서를 모두 적용한다.
